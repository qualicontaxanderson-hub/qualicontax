# 🚀 INSTRUÇÕES RÁPIDAS - Como Fazer Funcionar

## ⚡ 3 PASSOS SIMPLES

### 1️⃣ LIMPAR CACHE (2 minutos)

**Windows/Linux:**
1. Pressione `Ctrl + Shift + Delete`
2. Marque "Imagens e arquivos em cache"
3. Clique "Limpar dados"

**Mac:**
1. Pressione `Cmd + Shift + Delete`
2. Marque "Cache"
3. Clique "Limpar dados"

### 2️⃣ RECARREGAR PÁGINA (5 segundos)

1. Vá para: https://app.qualicontax.com.br/clientes/novo
2. Pressione `F5` ou `Ctrl + R` (Mac: `Cmd + R`)

### 3️⃣ TESTAR CNPJ (30 segundos)

1. Digite qualquer CNPJ válido
2. Clique "Consultar CNPJ"
3. Aguarde 2-3 segundos
4. Você verá um **ALERT** dizendo quantos campos foram preenchidos

## ✅ RESULTADO ESPERADO

Você verá um popup assim:

```
✅ Consulta CNPJ realizada com sucesso!

📊 RESUMO DO PREENCHIMENTO:
✅ Campos preenchidos: 12
⚠️ Campos não disponíveis: 5

Por favor, revise os dados preenchidos
e complete os campos restantes manualmente.
```

E os campos estarão preenchidos no formulário! 🎉

## ❌ SE NÃO FUNCIONAR

Se o alert mostrar **menos de 5 campos**, ele mesmo vai te avisar:

```
⚠️ ATENÇÃO: Poucos campos foram preenchidos!
Isso NÃO É NORMAL.

Possíveis causas:
1. Cache do navegador (MAIS PROVÁVEL)
2. Código não deployado

SOLUÇÃO: Feche o navegador completamente,
reabra e pressione Ctrl+Shift+Delete
para limpar o cache.
```

Siga as instruções do próprio alert!

## 🆘 AINDA NÃO FUNCIONA?

1. Feche o navegador **COMPLETAMENTE** (todas as abas, todas as janelas)
2. Reabra o navegador
3. Vá direto para: https://app.qualicontax.com.br/clientes/novo
4. Teste novamente
