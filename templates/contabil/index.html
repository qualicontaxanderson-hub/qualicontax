Sobre o Módulo Contábil

Importação OFX

Importe extratos bancários no formato OFX (Open Financial Exchange) diretamente do seu banco. O sistema processará automaticamente todas as transações.

Memorizações por Grupo

Configure regras de classificação que se aplicam a todos os clientes de um grupo. Ideal para empresas do mesmo ramo ou com padrões similares de movimentação.

Memorizações Individuais

Crie regras específicas para cada cliente. As memorizações individuais têm prioridade sobre as memorizações do grupo.

Exportação Contábil

Após classificar as transações, exporte os dados para diversos sistemas contábeis como Domínio, Alterdata, Sage, entre outros.
